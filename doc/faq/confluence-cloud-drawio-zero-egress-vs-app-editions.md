The draw.io app is available for Confluence Cloud in different versions offering different levels of security options for handling sensitive data in diagrams. 


| Features | [draw.io<br>app editions](https://marketplace.atlassian.com/apps/1210933/draw-io-diagrams-uml-bpmn-aws-erd-flowcharts?tab=pricing) | [draw.io<br>Zero Egress](https://marketplace.atlassian.com/apps/1235446/draw-io-zero-egress-diagramming-secured-on-forge)|
| :------- |           :------------:          |    :--------:       |
| Atlassian framework | Transitioning to Forge-only&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; | Forge-only |
| [Cloud Fortified](/blog/drawio-atlassian-cloud-fortified.html) | yes | yes |
| [Zero egress](/blog/drawio-forge-zero-egress.html) (``lockdown``) | advanced only | yes |
| [Data residency support](/blog/data-governance-lockdown.html) | yes | yes |
| [Gliffy import and mass import](/doc/faq/mass-import-gliffy-confluence-cloud.html) | yes | no |
| [Collaborative editing](/blog/collaborative-editing-confluence-cloud.html) | real time | interval |
| [Smart templates](/blog/smart-diagram-generation.html) | yes | no |
| [Mermaid diagrams](/blog/mermaid-diagrams.html) | yes | yes |
| [Dark mode](/blog/dark-mode-diagram-editor.html) & [adaptive colours](/blog/adaptive-colours.html) | yes | yes |
| [PDF export](/doc/faq/export-to-pdf.html) | yes | yes |
| [Custom editor configuration](/doc/faq/configure-diagram-editor.html) | yes | yes | 
| [Google Drive](/doc/faq/embed-diagram-googledrive-confluence-cloud.html) and [OneDrive](/doc/faq/embed-diagram-onedrive-confluence-cloud.html) | yes | no |

<br />
**Note:** The draw.io Zero Egress app will be available for Atlassian's Isolated Cloud and Government Cloud when they are released by Atlassian.
